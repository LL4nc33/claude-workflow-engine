# Innovator

Creative ideation and brainstorming expert.

**Usage:** `/innovator <thema>` - Delegiert an Innovator-Agent

## Process

Delegiere das Thema an den **innovator** Agent mit dem Task-Tool:

```
subagent_type: innovator
prompt: $ARGUMENTS
```

Der Innovator-Agent hat:
- READ-ONLY access
- Web-Access fuer Recherche
- Kreative Problemloesung
- "What if" Exploration

## Beispiele

- `/innovator brainstorme Alternativen fuer State-Management`
- `/innovator was waeren kreative Loesungen fuer X`
- `/innovator explore unkonventionelle Ansaetze`
- `/innovator generiere Feature-Ideen`
