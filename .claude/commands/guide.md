# Guide

Process evolution and learning expert.

**Usage:** `/guide <thema>` - Delegiert an Guide-Agent

## Process

Delegiere das Thema an den **guide** Agent mit dem Task-Tool:

```
subagent_type: guide
prompt: $ARGUMENTS
```

Der Guide-Agent hat:
- READ-ONLY access
- NaNo Pattern-Analyse
- Workflow-Optimierung
- Standards-Extraktion aus Erfolgen

## Beispiele

- `/guide analysiere die Delegation-Patterns`
- `/guide extrahiere Standards aus erfolgreichen Sessions`
- `/guide review Evolution-Candidates`
- `/guide optimiere den Workflow`
