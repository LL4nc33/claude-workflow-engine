# Product Mission

## What
A specialized multi-agent system for AI-assisted software development, combining spec-driven workflows with domain-specific expert agents.

## Why
To provide structured, standards-compliant development workflows where specialized agents (architect, debug, security, devops, etc.) collaborate through well-defined interfaces, reducing context loss and improving code quality.

## How
- 7 specialized agents with defined permission models
- Standards injection for consistent conventions
- Spec-driven workflow for complex feature development
- GDPR-compliant, EU-hosted operation

## Target Users
- Software developers using Claude Code
- Teams needing structured AI-assisted development
- EU-based organizations requiring data sovereignty
