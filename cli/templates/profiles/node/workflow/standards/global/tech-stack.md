# Tech Stack - Node.js

## Runtime
- **Node.js** >= 18 (LTS)
- **TypeScript** >= 5.3

## Package Management
- **npm** (default) or **pnpm**
- Always use lockfile (`package-lock.json` / `pnpm-lock.yaml`)
- Use `npm ci` in CI/CD (not `npm install`)

## Framework Options
- **Express** or **Fastify** for APIs
- **Next.js** for full-stack
- **NestJS** for enterprise

## Testing
- **Vitest** or **Jest** for unit tests
- **Supertest** for API integration tests
- **Playwright** for E2E

## Linting & Formatting
- **ESLint** with TypeScript parser
- **Prettier** for formatting
- **husky** + **lint-staged** for pre-commit

## Build
- **tsc** for libraries
- **esbuild** or **swc** for applications
- **tsup** for dual CJS/ESM packages
